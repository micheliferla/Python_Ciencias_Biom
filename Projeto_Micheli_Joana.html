Anlise comparativa da protena Citocromo B em espcies de Leishmania e outros organismos

Introduo e objetivo

Introduo:

A leishmaniose uma doena causada pelo parasita do gnero Leishmania

Citocromo B: protena que transporta eltrons e gera o gradiente de prtons para sntese de ATP

Em Leishmania, o gene cyt b est localizado no maxicrculo do cinetoplasto, tem cerca de 1080 pb e rico em AT

Escolhemos o gene da protena citocromo b:

  • altamente conservado entre eucariotos
  • amplamente disponvel em bases pblicas
  • apresenta tamanho relativamente constante

Objetivo:

Comparar sequncias da protena citocromo b em diferentes espcies de Leishmania, analisando suas caractersticas e relaes filogenticas com outros organismos

Olhando a lista printada acima, identificamos algumas caractersticas:

  • algumas espcies repetidas (queremos apenas uma sequncia por espcie).
  • algumas espcies indefinidas (sp. ou cf.).
  • uma sequncia com tamanho fora do padro (1060 - Leishmania sp.), possivelmente indicando um outlier ou uma sequncia incompleta.

O cdigo a seguir limpa o data set considerando os pontos acima. Especificamente:

  • remove sp., cf.
  • remove duplicadas.
  • remove sequncias com tamanho fora do normal.
In[4]:
new_records = []   # nova lista de records (substituir a antiga), inicialmente vazia
names_set = set()  # set com todos nomes, inicialmente vazio

for record in records: # Para cada record:

  # Pega nome da espcie usando get_species_name.
  species_name = get_species_name(record)

  # Pula espcies genricas (sp. ou cf.).
  if species_name.startswith("Leishmania sp") or species_name.startswith("Leishmania cf"):
    continue

  # Pula espcies repetidas.
  if species_name in names_set: # se species_name j foi visto, pula
    continue
  names_set.add(species_name) # (adiciona species_name ao set de nomes j vistos)

  # Pula sequncias com tamanho fora do range aceitvel, que definimos estar entre 1070 e 1090.
  if len(r.seq) < 1070 or len(r.seq) > 1090:
    continue

  # Ok - record passou os testes - adiciona  lista new_records.
  new_records.append(record)

# Definio de funo que recebe um record (sequncia) e calcula a porcentagem de GC.
def gc_percent(r):

Agora, importamos e inicializamos a funo PairwiseAligner do Biopython para fazer os alinhamentos. (Cdigo obtido da internet para referncia.)

In[10]:
from Bio.Align import PairwiseAligner

# Define estrutura PairwiseAligner que ser utilizada para alinhar.
align = PairwiseAligner()
align.mode = "global"
align.match_score = 1
align.mismatch_score = 0
align.open_gap_score = align.extend_gap_score = -1

# Definimos uma funo auxiliar que calcula o score do alinhamento entre sequncias a e b, em porcentagem relativa.
def align_score(a, b):
  return 100.0 * align.score(a.seq, b.seq) / len(a.seq)

Ento, criamos uma matriz (representado por uma lista de listas) e um pandas DataFrame que contm o alinhamento de todos pares de espcies.

In[11]:
# Cria matriz 2D (lista de listas) com todos os scores entre pares de records.
score_matrix = [[align_score(a, b) for a in records] for b in records]

# Converte para DataFrame.
df = pd.DataFrame(score_matrix)

Discusso sobre resultados.

A alta proporo de AT no gene citocromo b em Leishmania:

  • Provoca vis no uso de cdons e aumenta a frequncia de mutaes (A T), podendo interferir nas anlises filogenticas
  • Favorece inseres e delees (indels) devido instabilidade na replicao do DNA
  • Reduz a estabilidade qumica, dificultando o desenho e eficincia de primers especficos
